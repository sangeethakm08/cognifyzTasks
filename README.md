# cognifyzTasks
Cognifyz Full Stack Development Tasks

This repository contains all the tasks completed as part of the Cognifyz Full Stack Development Internship.
Each task focuses on core concepts of web development including HTML, CSS, JavaScript, Node.js, Express, EJS, server-side validation, and project deployment.

### Task 1 — Basic Form & Express Server

Key Learnings

Creating an HTML form

Handling POST requests with Node.js + Express

Understanding routing

Rendering responses using EJS

Using internal/inline CSS for styling

Deliverables

index.ejs form page

response.ejs result page

Basic server.js setup


### Task 2 — Inline Styles, Inline JS & Server-Side Validation

Key Learnings

Designing forms with inline CSS only

Implementing inline JavaScript (inside attributes)

Adding client-side validation

Adding server-side validation in Express

Storing user data in temporary in-memory storage

Rendering dynamic pages using EJS

Deliverables

Inline styled form (index.ejs)

Inline JavaScript validation

Validated data displayed using EJS

Clean Express server with validation logic

###  Task 3 — Advanced CSS Styling & Responsive Design
Key Learnings

Building multi-section page layouts with modern CSS

Applying advanced styling such as transitions, shadows, spacing, and hover effects

Implementing fully responsive design using Flexbox/Grid

Using media queries to optimize the layout across devices

Integrating a CSS framework like Bootstrap for consistent UI components

Deliverables

Fully styled and responsive webpage

Multi-section layout with improved visual hierarchy

CSS animations/transitions applied where needed

Bootstrap (or other CSS framework) integrated

Organized stylesheet for easy scalability
